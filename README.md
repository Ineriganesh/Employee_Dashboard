# Employee Management Dashboard

## Project Overview

The Employee Management Dashboard is a modern, responsive web application built using React. It provides a complete solution for managing employee records, including authentication, employee CRUD operations, image uploads with preview, search and filtering, and print functionality. The application follows clean UI/UX principles and works seamlessly across desktop and mobile devices.

---

## Tech Stack Used

- React (TypeScript)
- Vite
- Tailwind CSS
- shadcn-ui
- LocalStorage (for mock data persistence)

---

## Features

- Mock login authentication
- Protected dashboard routes
- Dashboard summary cards (Total, Active, Inactive employees)
- Add, Edit, Delete employee records
- Profile image upload with live preview
- Search employees by name
- Combined filters (Gender & Active/Inactive status)
- Print employee list
- Responsive UI for desktop and mobile
- Clean empty and loading states

## How to Run the Project Locally

### Prerequisites

- Node.js
- npm  
(Recommended: install using nvm)

### Steps

```bash
# Clone the repository
git clone https://github.com/Ineriganesh/Employee_Dashboard.git

# Navigate to the project directory
cd Employee_Dashboard

# Install dependencies
npm install

# Start the development server
npm run dev
